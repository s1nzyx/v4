Чтобы твой проект выглядел профессионально, а код и база данных работали без ошибок, имена файлов должны быть написаны латиницей, в нижнем регистре и без пробелов.

Поскольку в первом шаге в SQL-скрипт базы данных мы уже заложили определенные названия для товаров, давай структурируем абсолютно все изображения для папки `assets/images/`.

### 1. Системные баннеры (Интерфейс)

Эти фотографии будут отвечать за фоны и разметку страниц согласно макетам:

* **`banner_main.jpg`** — Фотография для фона блока «Краткая информация» на Главной странице (`index.php`).
* **`banner_auth.jpg`** — Жёлтый вертикальный баннер для страниц Авторизации (`login.php`) и Регистрации (`registration.php`).

### Как теперь вывести эти фотографии вместо текстовых заглушек в коде?

В коде прошлых шагов мы использовали CSS-заглушки (`card-image-placeholder`), чтобы сайт работал даже без картинок. Чтобы заменить их на реальные фотографии, сделай три простые правки:

#### А. На главной (`index.php`) и странице категорий (`product_list.php`)

Найди в коде этот блок:

```php
<div class="card-image-placeholder">
    ФОТО ТОВАРA
</div>

```

И замени его на вывод картинки из папки:

```php
<div class="card-image" style="height: 200px; overflow: hidden; background: #FFF9E3; display: flex; align-items: center; justify-content: center;">
    <img src="assets/images/<?= htmlspecialchars($product['image']) ?>" alt="<?= htmlspecialchars($product['name']) ?>" style="width: 100%; height: 100%; object-fit: cover;">
</div>

```

#### Б. В корзине (`basket.php`)

Найди строчку:

```php
<td><div class="basket-img-placeholder">ФОТО</div></td>

```

Замени на:

```php
<td>
    <img src="assets/images/<?= htmlspecialchars($item['id']) ?>.png" alt="" style="width: 60px; height: 60px; object-fit: cover; border-radius: 4px;">
</td>

```

*(Здесь мы динамически подставляем имя картинки, используя ID товара, например `1.png`, или можно сделать запрос и выводить `$item['image']`)*.

#### В. На страницах Входа и Регистрации (`login.php` / `registration.php`)

Найди блок баннера:

```html
<div class="auth-banner-side">
    <div class="banner-watermark">...</div>
</div>

```

Чтобы там была полноценная фотография (как ты просил в ТЗ), добавь её фоном прямо в тег:

```html
<div class="auth-banner-side" style="background-image: url('assets/images/banner_auth.jpg'); background-size: cover; background-position: center;">
    <div class="banner-watermark" style="font-size: 32px; color: rgba(0,0,0,0.15);">ДЭМОЭКЗАМЕН место для баннера</div>
</div>

```

Точно так же для фона краткой информации в `index.php`:

```html
<div class="main-banner" id="description" style="background-image: url('assets/images/banner_main.jpg'); background-size: cover; background-position: center;">

```

Теперь структура папки `assets/images/` полностью готова, логична и готова к демонстрации экспертам!
